# ESM with Bundle

This option is intended for use with modern frameworks like **React**, **Vue** or **Svelte**. The CSS is automatically injected into the page at runtime, so you only need to import a single file.

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs>

## Usage

Import `initOVA` and call it once the DOM is ready.

<Tabs>
  <TabItem value="react" label="React">

```js
import { initOVA } from "on-view-animations";
import { useEffect } from "react";

export default function App() {
  useEffect(() => {
    initOVA();
  }, []);

  return (
    <div data-ova="animation: fade-top; duration: 800ms">
      <h1>Hello World</h1>
    </div>
  );
}
```

  </TabItem>
  <TabItem value="vue" label="Vue">

```js
<script setup>
import { initOVA } from 'on-view-animations';
import { onMounted } from 'vue';

onMounted(() => {
  initOVA();
});
</script>

<template>
  <div data-ova="animation: fade-top; duration: 800ms">
    <h1>Hello World</h1>
  </div>
</template>
```

  </TabItem>
  <TabItem value="svelte" label="Svelte">

```js
<script>
  import { initOVA } from 'on-view-animations';
  import { onMount } from 'svelte';

  onMount(() => {
    initOVA();
  });
</script>

<div data-ova="animation: fade-top; duration: 800ms">
  <h1>Hello World</h1>
</div>
```

  </TabItem>
  <TabItem value="vanilla" label="Vanilla ESM">

```html
<script type="module">
  import { initOVA } from "on-view-animations";
  initOVA();
</script>
```

  </TabItem>
</Tabs>

:::tip Direct Download
If you installed via direct download, replace the import path with `'./path/to/ova.bundle.esm.js'`.
:::
:::info CSS auto-injected
With this bundle you **do not** need to import any CSS file separately — it is automatically injected at runtime.
:::

## HTML

Add the `data-ova` attribute to any element you want to animate:

```html
<div data-ova>Animated element</div>
```
